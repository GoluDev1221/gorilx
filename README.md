# GorilX

GorilX is an open-source Cyber Safety Awareness Progressive Web App (PWA)
designed to educate users about common cyber threats, online frauds, and
digital safety practices.

The project focuses on prevention, awareness, and first-response guidance
for everyday internet users, using simple language and real-world scenarios.

## Purpose

The primary objective of GorilX is public awareness and social good.

- No advertisements
- No user data collection
- No tracking or monetization
- Built for education, not profit

Any future premium cybersecurity tools developed under this initiative
will reinvest their revenue into development, maintenance, and
cyber safety–related social causes.

## Features

- Cyber safety tips (Do / Don’t, Myth vs Reality)
- Gamified quizzes with awareness certificates
- Emergency mode with immediate steps and helpline guidance
- Client-side password strength checker
- Offline-friendly Progressive Web App

## Disclaimer

This project is independently developed for awareness purposes only.
It does not claim official authority and does not replace government
cybercrime portals or law-enforcement services.

Users are encouraged to report incidents through official channels.

## Tech Stack

- Progressive Web App (PWA)
- Client-side logic with static content
- No paid APIs or third-party tracking
