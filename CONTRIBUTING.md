# Contributing to GorilX

Thank you for your interest in contributing to GorilX.

This project focuses on public cyber safety awareness and is built for social good.

## How You Can Contribute
- Improve or add cyber safety content
- Add quiz questions and explanations
- Improve UI/UX or accessibility
- Fix bugs or improve performance
- Improve translations (Hindi / English)

## Getting Started
1. Fork the repository
2. Create a new branch
3. Make your changes
4. Open a Pull Request with a clear description

No prior cybersecurity expertise is required.
